J - Pangu and Stones(Upsolved)

题意： 需要将所有石子合并成一堆，每次只能合并连续的[L, R]堆，求最小合并费用or不能。

Solution:

区间DP，大概就是模版题的升级版。然而我并不会区间DP= =
转移方程大概为:
$dp[i][j][k] = \min ( dp[i][t][1] + dp[t + 1][j][k - 1] ), i \leq t < j, k != 1$
$dp[i][j][1] = \min ( dp[i][j][t] ), l \leq t \leq r$
$dp[i][i][0] = 0$
